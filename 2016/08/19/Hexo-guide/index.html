2.服务器的机子，只能访问内网，不能访问外网。
3.npm 安装命令都要有外网访问。难道一个接一个下载依赖包？

  2. 6.答案很简单，用代理，把服务器上的npm，设置代理到可以访问外网的机子上（代理工具：ccproxy、fiddler等），即可解决问题！
  4. 9.#npm config set proxy http://proxy.company.com:8080
    10.#npm config set https-proxy http://proxy.company.com:8080
  6. 13.搞到以后，删除代理设置：
    14.#npm config delete proxy
    15.#npm config delete https-proxy
  7. 17.其它详细参数：https://npmjs.org/doc/config.html
  9. 20.附：直接给linux设置网络代理：
    21.#export http_proxy http://proxy.company.com:8080
    22.#export https_proxy http://proxy.company.com:8080
